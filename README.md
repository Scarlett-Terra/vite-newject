## 📦 功能介紹 (Features)

### 📌 1. 飲料商品列表展示
- 顯示可加入購物車的商品清單  
- 提供商品名稱、描述、價格等資訊  

### 📌 2. 元件加入購物車
- 點擊商品可加入購物車  
- 若商品已存在則累加數量  

### 📌 3. 商品數量增減
- 可調整商品數量（＋ / −）  
- 即時計算總金額  

### 📌 4. 刪除單一商品
- 從購物車中移除項目  

### 📌 5. 金額計算
- 動態更新：單項小計 + 購物車總金額  

### 📌 6. 簡易登入驗證（Demo）
- 前端驗證帳號密碼    
- 錯誤帳密提示
